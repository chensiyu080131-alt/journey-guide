## 第 7 页：市场机会

**市场规模**：
- 2024 百强县旅游收入 **1.9 万亿**，占全国近 3 成
- 县域接待量占全国 **60%+**
- 年增速 **12–15%**

**竞品空白**：
- 小红书：有流量无闭环
- 马蜂窝：有深度无规模
- AI 旅游产品：无一专注中国县城

**关键结论**：没有任何平台系统性解决"小城怎么玩"

## 第 9 页：技术方案

```
用户输入 → POI 检索(高德) → LLM 聚合生成 → 路线优化 → 前端渲染
```

| 层 | 选型 |
|----|------|
| 前端 | Next.js 14 + TypeScript + Tailwind |
| AI | DeepSeek / 通义千问 |
| 地图 | 高德地图 JS API |
| 部署 | Vercel |

**技术亮点**：RAG + 可信度分级（缓解 LLM 幻觉）
